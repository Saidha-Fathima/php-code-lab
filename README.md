# PHP Code Lab 📚

A structured PHP learning project containing beginner to intermediate level concepts with practical examples. This project is designed to help understand PHP through real coding exercises with a simple and interactive learning approach.

---

## 🚀 Project Overview

This project provides hands-on practice for core PHP concepts using real examples, forms, and database operations with a clean UI for better understanding.

It includes the following topics:

* String Functions in PHP
* Date Functions in PHP
* GET Method (Form Handling)
* POST Method (Form Handling)
* Arrays in PHP
* Database Concepts (CRUD + SQL Basics)

Each topic is designed with:

* HTML interface
* PHP backend logic
* Output display section
* Simple and clean UI for learning

---

## ⚙️ Requirements

To run this project locally, you need:

* WAMP Server (Windows)
* Apache Server
* PHP 7 or higher
* MySQL (for database operations)

---

## 🖥️ How to Run Using WAMP Server

1. Install WAMP Server from the official website
2. Start Apache server (and MySQL if needed)
3. Move the project folder into the `www` directory
4. Open your browser and go to:
   `http://localhost/php-code-lab/`
