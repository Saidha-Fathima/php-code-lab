Connect Database

Purpose: This PHP code is used to connect to a MySQL database.


Step-by-step Explanation:

  • $servername → Defines the server (localhost)
  • $username → MySQL username (root)
  • $password → Password (empty in XAMPP)
  • $dbname → Database name (testdb)
  • mysqli_connect() → Creates connection
  • if(!$conn) → Checks connection failure
  • mysqli_connect_error() → Displays error
  • else → Displays success message

Output:

If successful → connected successfully

If failed → connection error message

PHP Code

<?php
$servername="localhost";
$username="root";
$password="";
$dbname="testdb";

$conn=mysqli_connect($servername,$username,$password,$dbname);

if(!$conn)
die("connection failed".mysqli_connect_error());
else
echo "connected successfully";
?>
