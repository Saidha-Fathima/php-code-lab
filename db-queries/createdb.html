Create Database

Purpose: This PHP script is used to create a new database in MySQL.


Step-by-step Explanation:

  • mysqli_connect() → Connects to MySQL server
  • "localhost" → Server name
  • "root" → Default username
  • "" → Password (empty in XAMPP)
  • CREATE DATABASE → SQL command
  • mysqli_query() → Executes SQL query
  • mysqli_error() → Shows error if query fails
  • mysqli_close() → Closes connection

Output:

If successful → Database created

If error → Error message displayed

PHP Code

<?php
$conn=mysqli_connect("localhost","root","","");
$sql="create database testdb";

if(mysqli_query($conn,$sql))
echo "database created successfully";
else
echo "error creating database".mysqli_error($conn);

mysqli_close($conn);
?>
