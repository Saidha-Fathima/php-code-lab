Create Table

Purpose: This PHP code is used to create a table inside a MySQL database.


Step-by-step Explanation:

  • mysqli_connect() → Connects to database (testdb)
  • CREATE TABLE → SQL command to create table
  • student → Table name
  • id → Primary key with auto increment
  • name → Text field (required)
  • email → Unique field
  • reg_date → Stores timestamp automatically
  • mysqli_query() → Executes query
  • mysqli_error() → Shows error if failed
  • mysqli_close() → Closes connection

Output:

If successful → Table created successfully

If failed → Error message displayed

PHP Code

<?php
$conn=mysqli_connect("localhost","root","","testdb");

$sql="create table student(
id int(6) unsigned auto_increment primary key,
name varchar(25) not null,
email varchar(50) unique,
reg_date timestamp default current_timestamp
on update current_timestamp
)";

if(mysqli_query($conn,$sql))
echo "Table created successfully";
else
echo "error creating Table".mysqli_error($conn);

mysqli_close($conn);
?>
