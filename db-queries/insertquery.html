Insert Query

Purpose: This PHP code is used to insert a new record into a MySQL table.


Step-by-step Explanation:

  • mysqli_connect() → Connects to database (testdb)
  • INSERT INTO → SQL command to add data
  • student → Table name
  • (name, email) → Columns
  • VALUES() → Data to insert
  • mysqli_query() → Executes query
  • mysqli_error() → Shows error if failed
  • mysqli_close() → Closes connection

Output:

If successful → Row Inserted successfully

If failed → Error message displayed

PHP Code

<?php
$conn=mysqli_connect("localhost","root","","testdb");

$sql="insert into student(name, email)
values('Saidha','saidha@gmail.com')";

if(mysqli_query($conn,$sql))
echo "Row Inserted successfully";
else
echo "error inserting data in student".mysqli_error($conn);

mysqli_close($conn);
?>
