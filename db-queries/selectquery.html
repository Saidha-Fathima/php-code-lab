Select Query

Purpose: This PHP code is used to retrieve data from a MySQL table.


Step-by-step Explanation:

  • mysqli_connect() → Connects to database
  • SELECT → SQL command to fetch data
  • student → Table name
  • mysqli_query() → Executes query
  • mysqli_num_rows() → Checks if records exist
  • mysqli_fetch_assoc() → Fetches each row
  • while loop → Displays all rows

Output:

Displays ID, Name, and Email of all students

PHP Code

<?php
$conn=mysqli_connect("localhost","root","","testdb");

$sql="select id,name,email from student";
$result=mysqli_query($conn,$sql);

if(mysqli_num_rows($result)>0)
{
    while($row=mysqli_fetch_assoc($result))
    {
        echo "ID: " .$row['id'];
        echo " Name: " .$row['name'];
        echo " Email: " .$row['email'];
    }
}
else
{
    echo "No records found";
}
?>
